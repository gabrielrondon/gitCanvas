gitCanvas

gitCanvas is an open-source framework to work collaboratively with Business Model Canvas through git. The files are divided into the categories. It's a good way to learn managers to use git to produce documents they usually do otherwise and keep versioning of canvas through the evolution of the project.

Welcome to gitCanvas

gitCanvas was created to be used in git courses. The objective is to replace traditional spreadsheets templates or presentations of Business Model Canvas which most of the managers and entrepreneurs use nowadays. With gitCanvas they have a way to learn git by working with the building of their canvas, so they can discover the advantages of git & github as versioning for instance.

Canvas and validation are an endless circle

Each validation of the hypothesis of a model demands a change in the canvas, so many contributors can change using their terminal and keep the versioning =)

gitCanvas is a simple application for ManTech
